Reviewer Recommender Options

This extension requires a GitHub Personal Access Token (PAT) to fetch pull request data and suggest reviewers.
Your token is stored securely using Chrome's storage and is only used to communicate with the GitHub API.

GitHub Personal Access Token (PAT)

You can create a PAT here. The token needs the following scopes:

  • repo (full control of private repositories) - Recommended for full functionality.
  • OR public_repo (read access to public repositories) - For public repositories only.
It's recommended to set an expiration date for your token for security.
